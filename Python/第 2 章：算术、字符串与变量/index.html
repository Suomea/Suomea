    交互式 Shell

    在命令行直接输出 python 即可打开 Python 的交互式 shell。

    hello world! 

    >>> print("hello world!")
hello world!

    算术运算

    Python 支持四种基本算术运算。 ， 还支持使用 ** 和 % 进行乘方和求余运算，支持使用// 进行整除运算。

    可以直接在交互式 shell 中进行 

    >>> 1+2
3
>>> 1-2
-1
>>> 2*3
6
>>> 3/2
1.5
>>> 3//2
1
>>> 2**2
4
>>> 3%2
1

    在 Python 中浮点数是带小数的数字，所有上述适用于整数的运算也适用于浮点数。 

    >>> 2.2**3
10.648000000000003

    浮点数3. 等于3.0，.5等于0.5。可见 Python 表述浮点数非常灵活。还可以使用科学计数法表示浮点数。需要注意的是区分3和3.0是很重要的，因为它们的内部表示大相庭径。 

    >>> 2e2
200.0
>>> 2e-2
0.02
>>> .34
0.34
>>> 3.
3.0
>>>

    浮点数存在上限和下限，即浮点数的运算可能存在溢出。

    字符串

    字符串有三种表示方式。单引号能够在字符串中包含双引号字符，双引号能够在字符串中包含单引号字符。三引号主要用来包含多行字符串，内部可以包含单引号字符和双引号字符。 

    "hello world"  

'hello world'  

"""  
hello world  
你好世界！  
"""

    字符串长度使用 len(s) 函数获得，返回一个整数。

    字符串拼接使用+字符。 

    >>> print('hello' + " world!")
hello world!
>>> print(2 * 'hello' + 2 * '!!')
hellohello!!!!
>>> print(len(2 * 'hello' + 2 * '!!'))
14
>>> len('aa') * '12'
'1212'

    类型转换

    整数转换为浮点数，字符串转换为浮点数。 

    >>> float(3)
3.0
>>> float('3')
3.0
>>> float('3.12')
3.12

    浮点数和整数转换为字符串。 

    >>> str(3)
'3'
>>> str(-2.3)
'-2.3'

    字符串和浮点数转换为整数。 

    >>> int('3')
3
>>> round(float('3.2'))
3
>>> round(float('3.6'))
4

    变量

    变量指向一个值，遇到变量时 Python 将其替换为指向的值。 

    >>> name = 'jacky'
>>> 2 * name
'jackyjacky'
>>> l = len(2 * name)
>>> l
10
>>> name
'jacky'

    变量的命名规则需要遵守以下条件： 1. 变量名的长度不受限制，但是必须是字幕、数字或者下划线。 2. 变量名的第一个字符不能是数字。 3. 变量名区分大小写。 4. 不能使用 Python 的关键字作为变量名。

    可以将任何值赋值给变量，包括其它变量的值（即使类型不同）。 

    >>> name
2
>>> name='jacky'
>>> name
'jacky'

    赋值就像是指针操作一样，没有复制。对于没有任何变量指向的值，Python 将自动将其删除。

    数字和字符串是不可变的。

    多重赋值，能够给多个变量同时赋值（并不常用）。 

    >>> a, b, c = 1, '2', 3.3
>>> a
1
>>> a, c, b
(1, 3.3, '2')
>>> a, b, c = c, b, a
>>> a, c, b
(3.3, 1, '2')
